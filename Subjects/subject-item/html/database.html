About the Course

Database is a course that teaches you how to store, organize, and manage data efficiently. You will learn how databases work, how to design them using ERD, and how to interact with data using SQL. It’s a core subject for anyone studying IT or working with data.

What You Will Learn

  • Understand what a database is and why it is used.
  • Learn the difference between relational and non-relational databases.
  • Understand the structure of relational databases (tables, rows, columns, keys).
  • Learn how to design an Entity-Relationship Diagram (ERD) and identify entities, attributes, and relationships.
